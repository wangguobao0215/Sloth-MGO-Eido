## CH1: 微信公众号

### 基础限制

| 参数 | 规范 | 备注 |
|------|------|------|
| 标题字数 | **最多64个字符**（含标点） | 建议控制在22字以内，手机端两行显示临界点 |
| 摘要/描述 | **最多120个字符** | 未填写时默认取正文前54字 |
| 封面图（头条） | **900 × 383 px**（2.35:1） | 推荐PNG/JPG，≤2MB |
| 封面图（次条） | **500 × 500 px**（1:1）或 200×200px | 次条显示更小 |
| 正文字数 | 无硬性上限，推荐**1500-4000字** | 超过4000字打开率下降 |
| 正文图片 | 宽度建议**1080px**，高度不限 | 单张≤10MB，GIF≤10MB |
| 正文图片数量 | 无上限，推荐**5-10张** | 每800字配1张图 |
| 阅读原文链接 | 1个 | 放在文末，用于导流 |
| 自定义菜单 | 最多**3个一级菜单**，每个下5个子菜单 | — |
| 推送频次 | 订阅号每日1次，服务号每月4次 | 服务号单次可发8条 |

### 内容规范

| 要素 | 要求 |
|------|------|
| 标题公式 | [数字/悬念/痛点] + [核心价值] + [目标读者] |
| 首段 | 150字以内点明核心价值，移动端首屏完整展示 |
| 段落长度 | 每段3-5行，避免大段文字堆砌 |
| 字号 | 正文建议15-16px，标注14px |
| 行间距 | 1.75-2倍行距 |
| 字间距 | 0.5-1px |
| 配色 | 正文色#333333，辅助色#888888，强调色用品牌色 |
| CTA位置 | 文中1处+文末1处，不超过2个 |
| 原创声明 | B2B内容建议开启原创，增加搜索权重 |

### SEO/搜索优化

- 标题包含核心关键词（微信搜一搜权重因素）
- 正文前200字包含关键词2-3次
- 使用话题标签（#标签#）提升搜一搜可见性

## CH2: 知乎

### 基础限制

| 参数 | 规范 | 备注 |
|------|------|------|
| 回答字数 | 无上限，推荐**1000-3000字** | 长回答获赞率更高 |
| 专栏文章字数 | 无上限，推荐**2000-5000字** | — |
| 标题字数（专栏文章） | **最多100字** | 建议25-40字最优 |
| 封面图（专栏） | **690 × 388 px**（16:9） | 推荐高清图 |
| 正文图片 | 宽度建议**720px** | — |
| 话题标签 | 每篇**最多5个话题** | 选择关注量10万+的话题 |
| 个人简介 | **最多100字** | 可放公众号引导 |
| 专栏名称 | **最多20字** | — |

### 内容规范

| 要素 | 要求 |
|------|------|
| 回答开头 | 先亮结论（知乎用户扫读习惯），再展开论述 |
| 内容结构 | 总→分→总；使用加粗标记核心观点 |
| 引用规范 | 数据必须标注来源，使用知乎引用格式 |
| 利益相关声明 | 回答中涉及自家产品时，开头声明"利益相关：XX公司XX" |
| 导流方式 | 文末"想了解更多，可以看我的专栏/公众号" — 避免硬广 |
| 互动 | 回答后24小时内回复前5条评论 |

### 注意事项
- 知乎对营销内容监控严格，账号频繁发广告可能被限流/封禁
- 回答优于发文章（回答有问题页面流量加持）
- 优先选择**关注量1万+、浏览量10万+**的问题回答

## CH3: 头条/百家号

### 基础限制

| 参数 | 头条号规范 | 百家号规范 |
|------|-----------|-----------|
| 标题字数 | **5-30字**（推荐20-25字） | **9-30字** |
| 正文字数 | 推荐**800-2000字** | 推荐**800-2000字** |
| 封面图 | **16:9**比例，≥600×360px | **16:9**，推荐1280×720px |
| 正文图片 | ≥400px宽度 | ≥200px宽度 |
| 话题/标签 | 最多**3个标签** | 最多**3个标签** |
| 发文频率 | 每日最多**5篇** | 新手期每日1篇 |
| 原创标记 | 需开通原创功能 | 需开通原创功能 |

### 内容规范

| 要素 | 要求 |
|------|------|
| 标题 | 信息量大，包含关键词，不做标题党 |
| 首段 | 100字以内说明文章价值，算法会抓取首段判断内容质量 |
| 内容密度 | 每200字一个核心信息点，配图辅助 |
| 小标题 | 使用H2/H3小标题，提升可读性+算法识别 |
| 导流限制 | **禁止正文内放外链/二维码**（会被限流） |
| 合规 | 必须标明广告内容（如为推广） |

### 推荐算法适配
- 标题关键词匹配推荐池（选择搜索量大的行业词）
- 完读率>50%可获更多推荐 → 控制文章长度
- 互动率（评论/收藏/转发）影响推荐权重 → 文末设引导

## CH4: 小红书

### 基础限制

| 参数 | 规范 | 备注 |
|------|------|------|
| 标题字数 | **最多20个字** | 必须包含关键词 |
| 正文字数 | **最多1000字** | 建议300-800字 |
| 话题标签 | **3-5个**，用#号 | 选择搜索量大+竞争适中的标签 |
| 图片数量 | **1-18张** | 推荐6-9张，信息量充足 |
| 图片尺寸 | **3:4**（1080×1440px）最优 | 也支持1:1、4:3 |
| 图片格式 | JPG/PNG，单张≤20MB | 注意画质清晰 |
| 视频时长 | **15秒-15分钟** | 推荐60-90秒 |
| 视频尺寸 | **9:16**竖屏（1080×1920px） | — |
| 个人简介 | 最多**100字** | 可放职业标签和联系方式暗示 |

### 内容规范

| 要素 | 要求 |
|------|------|
| 封面图 | 信息量大，文字清晰，使用高对比色彩，**是否被点击的决定因素** |
| 标题 | 包含核心关键词 + 数字/emoji + 结果承诺 |
| 正文开头 | 第一句用emoji/符号吸引注意，快速说明价值 |
| 正文格式 | 使用emoji分段、竖排清单体、每段2-3行 |
| 图片设计 | 统一视觉风格、信息图>纯装饰图、可加文字水印 |
| CTA | "评论区留言XX""点击主页了解更多" |
| 互动引导 | 结尾必须有互动话术："你遇到过类似问题吗？" |

### B2B适配建议
- 内容角度：实用干货 > 品牌宣传（如"B2B获客的5个隐藏技巧"）
- 人设定位：行业专家/资深从业者，非品牌官方号
- 注意事项：小红书对硬广敏感，B2B内容以知识分享切入

## CH5: LinkedIn

### 基础限制

| 参数 | 规范 | 备注 |
|------|------|------|
| 帖子（Post）字数 | **最多3000字符** | 建议800-1500字符最优 |
| 文章（Article）字数 | **最多12万字符** | 推荐1500-3000字 |
| 文章标题 | **最多150字符** | 建议40-70字符 |
| 文章封面图 | **1200 × 628 px** | 推荐16:9比例 |
| 帖子图片 | **1200 × 1200 px**（1:1）或 1200×628（1.91:1） | 单图/多图轮播 |
| 轮播文档（PDF） | **最多300页**，≤100MB | 推荐10-15页，竖版 |
| 视频 | 3秒-10分钟，≤5GB | 推荐30-90秒 |
| Hashtag | **3-5个** | 不超过5个 |

### 内容规范

| 要素 | 要求 |
|------|------|
| 帖子首行 | 钩子文案：用数据/问题/反常识开头（展开前只显示前3行） |
| 帖子格式 | 短段落（1-2句/段）、空行分隔、使用符号列表 |
| 个人帖 vs 公司页 | 个人帖互动率5-10x高于公司页 → B2B优先个人帖 |
| 互动时间 | 发布后1小时内回复每条评论（算法加权） |
| 发布时间 | 周二至周四，上午8-10点（目标时区） |
| 标签 | 3个最优：1行业词 + 1话题词 + 1品牌词 |
| 语言 | 英文为主（全球受众），中文（中国区受众） |
| 轮播PDF | 封面页醒目 + 每页一个核心观点 + 末页CTA |

### LinkedIn文章（长文）

| 要素 | 要求 |
|------|------|
| 标题 | 40-70字符，包含关键词 |
| 封面图 | 必须设置，1200×628px |
| 正文结构 | H2小标题分段 + 数据引用 + 个人见解 |
| CTA | 文末一个软性CTA（如"你怎么看？"） |

## CH6: EDM 邮件

### 基础限制

| 参数 | 规范 | 备注 |
|------|------|------|
| 主题行字数 | **50字以内**（中文约25字） | 移动端显示约30字，确保核心信息前置 |
| 预览文本 | **80字以内** | 补充主题行，不要重复 |
| 发件人名 | **真人名字**（如"王磊 | XX公司"） | 避免纯公司名 |
| 正文宽度 | **600px**（HTML模板宽度） | 确保移动端适配 |
| 正文字数 | **150-500字/封** | 精简，一封一主题 |
| 图片数量 | **2-5张** | 控制图文比，纯图邮件易进垃圾箱 |
| 图片 | 总图片≤**800KB** | 避免大图加载慢 |
| CTA按钮 | **1个主CTA** | 颜色突出，文案6字以内 |
| 退订链接 | **必须有** | PIPL合规要求 |
| 发件频率 | 培育序列：**每3-7天1封** | 过频=退订率上升 |

### 主题行规范

| 规则 | 说明 |
|------|------|
| 避免垃圾词 | "免费""促销""点击这里""恭喜""赚钱" → 触发垃圾过滤 |
| 个性化 | 使用{{FirstName}}变量，提升20%+打开率 |
| A/B测试 | 每次发送测试2个主题行，样本≥500封/组 |
| 预发送检查 | Spam Score检测（目标<5分），多客户端预览测试 |

### 技术要求

| 技术项 | 要求 |
|--------|------|
| 发送认证 | SPF + DKIM + DMARC 三项认证 |
| 退信处理 | 硬退信（Hard Bounce）自动移出列表 |
| 模板编码 | HTML Table布局（非Div），兼容Outlook |
| 暗色模式 | 适配Dark Mode，避免白背景文字消失 |
| 追踪像素 | 1×1 tracking pixel 用于打开追踪 |
| 链接追踪 | 所有链接UTM标记 |

## CH8: 视频号/抖音（短视频）

### 基础限制

| 参数 | 视频号规范 | 抖音规范 |
|------|-----------|---------|
| 视频时长 | 1秒-30分钟（推荐60-180秒） | 15秒/60秒/3分钟/15分钟 |
| 视频尺寸 | **9:16竖屏**（1080×1920px） | **9:16竖屏**（1080×1920px） |
| 标题/描述 | **最多1000字** | **最多55字**（重要！） |
| 话题标签 | **3-5个** | **3-5个** |
| 封面图 | 从视频中选帧或上传 | **9:16**自定义封面 |
| 视频大小 | ≤4GB | ≤4GB |
| 帧率 | 推荐30fps | 推荐30fps |

### 短视频脚本结构

#### 15秒脚本（适合抖音信息流/视频号碎片内容）

```
[0-3秒] 钩子 Hook
  → 冲突/问题/数据冲击，留住观众
  模板："做B2B营销最怕什么？——花了钱，一个线索没有。"

[3-10秒] 核心信息 Core
  → 一个关键观点/方法/事实
  模板："问题出在[原因]。正确的做法是[解决方案]。"

[10-15秒] CTA
  → 引导关注/评论/点击链接
  模板："更详细的方法论在主页置顶视频里。关注我，别错过。"
```

#### 60秒脚本（适合干货分享/案例展示）

```
[0-5秒] 钩子 Hook
  → 强冲突/反常识/痛点共鸣
  
[5-15秒] 问题展开 Problem
  → 痛点具体化，观众代入感

[15-45秒] 解决方案 Solution
  → 分步骤/清单展示方法
  → 使用文字弹幕/画面辅助
  
[45-55秒] 案例/证据 Proof
  → 快速展示数据/截图/客户评价

[55-60秒] CTA
  → "关注+评论，下期讲[话题]"
```

#### 3分钟脚本（适合深度讲解/产品演示）

```
[0-10秒] 钩子+议题预告
[10-30秒] 背景说明+痛点描述
[30-120秒] 核心内容（3个要点/步骤）
[120-150秒] 案例/数据佐证
[150-170秒] 总结+金句
[170-180秒] CTA+预告下期
```

### 制作规范

| 要素 | 要求 |
|------|------|
| 画质 | 1080p最低，4K更优 |
| 字幕 | **必须添加**（90%用户静音观看） |
| 字幕样式 | 白色字+黑色描边，字号≥24pt |
| BGM | 使用平台版权库音乐，音量低于人声 |
| 出镜人 | B2B推荐真人出镜（信任感），统一形象 |
| 画面稳定 | 使用三脚架/稳定器 |
| 安全区域 | 顶部15%、底部20%不放核心信息（被UI遮挡） |

## CH9: SEM/信息流广告文案规范

### SEM搜索广告

| 参数 | 百度搜索广告 | Google Ads |
|------|-------------|-----------|
| 标题长度 | **最多50字**（含标点） | 30字符×3条标题 |
| 描述长度 | **最多80字**×2条 | 90字符×2条描述 |
| 显示URL | 域名+路径（各15字） | 域名+路径（各15字符） |
| 附加信息 | 子链接/电话/地址/结构化摘要 | Sitelinks/Callouts/Snippets |

### SEM文案模板

```
标题公式（三段式）：
  标题1：[核心关键词] + [价值承诺]
  标题2：[差异化优势/数据]
  标题3：[CTA]

示例：
  标题1：B2B营销自动化平台 - 降低50%获客成本
  标题2：已服务500+企业客户 | 免费试用
  标题3：立即预约产品演示

描述公式：
  描述1：[痛点+解决方案] + [核心功能]
  描述2：[社会证明] + [CTA+降低门槛]

示例：
  描述1：告别手动管理线索，一站式完成从获客到转化的全链路管理。支持多渠道线索归集、智能评分、自动化培育。
  描述2：服务覆盖制造、科技、金融等行业。注册即享14天免费试用，无需信用卡。
```

### 信息流广告

| 参数 | 朋友圈广告 | 头条信息流 | 百度信息流 |
|------|-----------|-----------|-----------|
| 外层文案 | **最多40字**（建议25字） | **最多30字** | **最多30字** |
| 外层图片 | 横版800×640 / 竖版640×800 | 690×388/1280×720 | 1200×628 |
| 外层视频 | 6-30秒 | 6-60秒 | 6-30秒 |
| 落地页标题 | **最多17字**（手机端首屏） | — | — |
| 落地页表单 | ≤5个字段（姓名/手机/公司/职位/需求） | ≤5字段 | ≤5字段 |

### 落地页（Landing Page）规范

| 要素 | 要求 |
|------|------|
| 首屏 | 价值主张+CTA按钮，首屏即可转化 |
| 加载速度 | ≤3秒（移动端） |
| 表单字段 | ≤5个，姓名+手机/邮箱必填 |
| 信任元素 | 客户Logo墙、安全认证标识、媒体背书 |
| 移动适配 | 响应式设计，CTA按钮拇指易点击（≥44×44px） |
| UTM追踪 | 每个落地页URL带完整UTM参数 |
| 隐私合规 | 表单旁勾选隐私政策同意框 |

### 广告合规检查（叠加compliance-rules.md）

```
广告文案合规检查清单：
□ 无绝对化用语（最好/第一/唯一等）
□ 数据声明有来源
□ 未使用竞品品牌词（除非有合法授权）
□ 落地页内容与广告文案一致（不误导）
□ 价格信息真实（非虚构原价）
□ 隐私政策链接可访问
□ 表单同意框未预勾选
```
