## 三、阶段内容匹配规则

### 3.1 认知阶段（Awareness）

> 对应序列 Step 1-2，线索尚未意识到自身痛点或刚开始探索。

| 维度 | 规则 |
|---|---|
| 内容调性 | 教育性、启发性，不推销 |
| 适合内容类型 | 行业趋势报告、调研数据、思想领导力文章、痛点解析 |
| 不适合内容类型 | 产品功能介绍、定价信息、产品演示 |
| CTA 风格 | 柔性CTA：阅读更多、下载报告、关注公众号 |
| 推送频率 | 每3-5天一次 |
| 关键指标 | 邮件打开率、内容阅读时长、二次访问率 |

### 3.2 考虑阶段（Consideration）

> 对应序列 Step 3-4，线索已认识到痛点，开始评估解决方案。

| 维度 | 规则 |
|---|---|
| 内容调性 | 专业性、对比性，帮助客户建立评估框架 |
| 适合内容类型 | 产品对比指南、解决方案白皮书、客户案例、ROI计算器 |
| 不适合内容类型 | 过于宏观的行业趋势（已过了这个阶段） |
| CTA 风格 | 中性CTA：下载方案、查看案例、使用计算器 |
| 推送频率 | 每3-4天一次 |
| 关键指标 | 内容下载量、高价值页面访问（定价页、案例页）、表单填写 |

### 3.3 决策阶段（Decision）

> 对应序列 Step 5-7，线索已有明确需求，进入供应商选择阶段。

| 维度 | 规则 |
|---|---|
| 内容调性 | 紧迫性、确定性，推动行动 |
| 适合内容类型 | 产品演示邀约、免费试用、专家咨询、客户证言视频、实施路线图 |
| 不适合内容类型 | 泛泛的行业内容（客户已经不需要被教育） |
| CTA 风格 | 强CTA：立即预约、申请试用、联系专家 |
| 推送频率 | 按触发条件实时推送 |
| 关键指标 | 演示预约率、试用申请率、销售对话率 |

## 四、序列退出条件

| 退出场景 | 触发条件 | 处理方式 |
|---|---|---|
| 正向毕业 | 线索评分升至H级（>=80分）或产生明确购买信号 | 立即退出序列，48h内交付销售，附带完整互动历史 |
| 提前升级 | 线索在序列中期就产生强行为信号（如主动申请演示） | 跳过后续步骤，直接升级交付销售 |
| 培育成功 | 线索从B级升至A级 | 退出当前序列，进入A级专属培育序列（更短、更直接） |
| 无响应退出 | 连续{N}步（建议3步）无任何响应 | 退出序列，标记为"培育无响应"，转入冷线索池 |
| 主动退订 | 客户回复退订或标记为垃圾邮件 | 立即停止所有推送，标记为"已退订"，尊重客户意愿 |
| 重复线索 | 发现CRM中已有该客户的活跃商机 | 退出序列，避免市场和销售同时触达造成混乱 |
| 信息过时 | 联系人离职或企业发生重大变化 | 暂停序列，更新信息后重新评估是否继续 |
